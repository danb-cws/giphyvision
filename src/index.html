GiphyVision

GiphyVision uses your camera and image recognition to return relevant gifs from the objects around you.

GiphyVision is powered by:

  • Giphy
  • Google Cloud Vision
  • Heroku
  • Firebase

GiphyVision is a learning project by Dan Brown, including use of Webpack, ES6, node, RTC camera, Google Cloud Vision API and Giphy API.
